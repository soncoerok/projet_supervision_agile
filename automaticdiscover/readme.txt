Les deux fichier shell permettent de lancer le programme. 
	* install.sh : télécharge les sources, compile et lance le programme (maven requis)
	* install-without-mvn.sh : télécharge le projet compilé et lance le programme

Dans le cas d'une compilation des sources, le premier lancement sera un peu long puisque maven téléchargera toutes les dépendances.

Nous avons réalisé l'ensemble des fonctionnalités demandées lors de la première réunion SCRUM.
Etant donné que le fonctionnel demandé pour le premier sprint était assez léger, nous avons axé le temps restant sur l'outillage ainsi que l'IHM pour permettre de partir sur de bonnes bases pour le sprint 2.
